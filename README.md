# Guardian of Dimensions
2025-1 Immersive Media Programming AR Team Project

차원의 균열로 인해 현실 세계에 침입한 마법 생명체를 플레이어가 차원의 수호자가 되어 물리치는 콘셉트의 AR 액션 게임입니다.

플레이어는 AR 환경에서 손 동작을 인식해 공격 방식을 선택하고, 보스의 공격 패턴을 피하며 전투를 진행합니다. 보스는 페이즈에 따라 다른 공격 패턴과 피해량을 가지며, 플레이어는 상황에 맞춰 공격과 회피를 반복하며 보스를 처치해야 합니다.

### Tech Stack
- Engine: Unity
- Language: C#
- Framework: AR Foundation, ARCore
- Platform: Android (AR)
